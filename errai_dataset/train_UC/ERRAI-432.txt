Erase LogAdapter; replace usages with java.util logging
GWT implements java.util logging (see https://developers.google.com/web-toolkit/doc/latest/DevGuideLogging) which works on both client and server. You can opt out of client-side logging by not inheriting com.google.gwt.logging.Logging, or by setting some properties:

    <set-property name="gwt.logging.logLevel" value="SEVERE"/>          # To change the default logLevel
    <set-property name="gwt.logging.enabled" value="FALSE"/>            # To disable logging
    <set-property name="gwt.logging.consoleHandler" value="DISABLED"/>  # To disable a default Handler

when logging is disabled, it doesn't contribute any weight to the javascript.
