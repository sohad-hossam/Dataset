Note: the original description of this issue made the assumption that this error was caused by an invalid field name for the event target. This is not the case; even with the correct field name, the error persists.

Attempting to compile this class:

@Templated("#root")
public class WelcomePage extends CompositePage {

  @Inject @DataField Button startButton;
  
  @EventHandler("startButton")
  public void onStartButtonPress(ClickEvent e) {
    System.out.println("You pressed the start button");
  }
