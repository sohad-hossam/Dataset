Application fails to start when logging is disabled.
On starting of the application in the browser, the application fails to start with the javascript error: Cannot read property 'length' of null'

If the application GWT module contains the following statement
{code}
<set-property name="gwt.logging.enabled" value="FALSE"/>
{code}

The LoggingHandlerConfigurator.java file (line:48 onwards)
{code}
...

  for (Handler logHandler : logHandlers) {
    if (logHandler instanceof HasWidgetsLogHandler) {
      logger.removeHandler(logHandler);
      ((HasWidgetsLogHandler)logHandler).clear();
    }
  }
