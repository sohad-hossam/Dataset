CDI: Duplicate local variable when overriding method injection
Given (for example):-
{code}
class Parent {

    Child child;

    @Inject
    void setChild( Child child ) {
        this.child = child;
    }

}
{code}
And
{code}
class NewParent extends Parent {

    @Inject
    @Override
    void setChild( @SomeQualifier Child child ) {
        super.setChild( child );
    }

}
{code}

Changing the sub-class to the following solves the problem (the only difference is that the method parameter name was changed from {{child}} to {{c}}):-
{code}
class NewParent extends Parent {

    @Inject
    @Override
    void setChild( @SomeQualifier Child c ) {
        super.setChild( c );
    }

}
{code}
